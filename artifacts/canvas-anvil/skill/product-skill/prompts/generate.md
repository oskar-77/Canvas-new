# Product Generation Prompt

Build one final image-generation prompt for a single product visual.

## Requirements

- use the collected product fields
- keep the image single-frame
- keep the product subject explicit
- keep the composition, lighting, and background explicit
- keep the text content and selling points explicit
- avoid adding unsupported facts or claims

## Missing Data Rule

If any required field is missing, ask follow-up questions before producing the final prompt.
