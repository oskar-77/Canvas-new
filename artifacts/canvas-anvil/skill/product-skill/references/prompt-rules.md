# Prompt Rules

Use these rules when building the final product visual generation prompt.

- Keep the output focused on one product image.
- State the intended size or aspect ratio.
- State the visual style clearly.
- State the color direction clearly.
- State the composition, focal area, and lighting clearly.
- State the background direction clearly.
- Include the product subject explicitly.
- Include the provided copy and selling points explicitly.
- Use concise, concrete visual language.
- Prioritize product visibility, clarity, and hierarchy over decorative effects.
- Do not invent extra claims, specs, pricing, certifications, or dates.
- If the user provides a reference image, treat it as style, composition, lighting, or mood guidance.
- Do not treat a reference image as a source of facts.
- If the reference image conflicts with explicit user instructions, follow the explicit user instructions.
- If a product image is provided, treat it as the primary subject reference and preserve the intended product identity.
