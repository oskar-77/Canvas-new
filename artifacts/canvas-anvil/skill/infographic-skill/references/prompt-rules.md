# Prompt Rules

Use these rules when building the final infographic generation prompt.

- Keep the output focused on one infographic image.
- State the intended size or aspect ratio.
- State the visual style clearly.
- State the color direction clearly.
- State the information hierarchy clearly.
- State chart intent when charting is requested.
- Include the required textual content and data points explicitly.
- Use concise, concrete visual language.
- Prioritize readability, grouping, and section clarity over decorative effects.
- Do not invent extra metrics, claims, rankings, dates, or brand facts.
- If the user provides a reference image, treat it as style, composition, spacing, or mood guidance.
- Do not treat a reference image as a source of facts or data.
- If the reference image conflicts with explicit user instructions, follow the explicit user instructions.
