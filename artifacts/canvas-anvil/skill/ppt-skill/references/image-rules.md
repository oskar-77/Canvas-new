# Image Rules

- Render slides as `16:9` presentation images unless the user explicitly asks for another aspect ratio.
- Keep text readable at normal presentation scale.
- Treat template images as style references only.
- Treat uploaded material images as optional supporting assets unless the user explicitly requires them.
- Do not invent factual claims, data, labels, or logos from reference images.
- Fill dead space with purposeful structure or decoration, not random clutter.
- Keep the generated result suitable for presentation delivery, not social-media poster styling by default.
