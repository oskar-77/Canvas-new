# Task Types

## `template_only`

Generate one or more reusable PPT template background images.

## `deck_from_idea`

Start from a loose topic, goal, or audience brief and produce a full presentation plan, then render slides.

## `deck_from_outline`

Start from a user-provided outline and convert it into a full presentation plan, then render slides.

## `deck_from_documents`

Start from uploaded or pasted source material, extract the usable facts, and convert them into a presentation plan, then render slides.

## `deck_edit_routing`

Map revision feedback to existing slide ids and classify edit types.

## `export_only`

Take existing rendered slides or prepared deck data and export in one or more supported delivery modes.
