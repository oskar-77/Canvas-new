# Template Contract

- Template output is a reusable slide background, not a completed content slide.
- Default aspect ratio is `16:9`.
- Templates should remain text-free unless the user explicitly requests branded text baked into the background.
- A template may define:
  - color direction
  - composition direction
  - visual density
  - decorative motifs
  - image treatment
- A template should not assume specific slide titles or bullet copy.
- Template generation may produce multiple variants when the user asks for options.
