# Plan Schema

Every slide in a full presentation plan must include:

- `id`
- `title`
- `content`
- `description`
- `layout`
- `note`

## Field meaning

- `id`: stable slide identifier
- `title`: short visible slide title
- `content`: audience-facing bullet points or short lines
- `description`: visual-generation instruction for the slide image
- `layout`: controlled layout hint
- `note`: speaker note or presenter reminder

## Rules

- `content` should be concise and presentation-friendly
- `description` should specify subject, composition, style, and visual emphasis
- `note` must always exist, even if short
- avoid overlong decks unless user intent clearly requires them
