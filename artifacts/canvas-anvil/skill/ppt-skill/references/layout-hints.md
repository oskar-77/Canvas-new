# Layout Hints

Use controlled layout hints instead of inventing arbitrary labels.

Recommended set:

- `cover`
- `title_bullets`
- `title_content`
- `two_column`
- `left_text_right_image`
- `right_text_left_image`
- `full_visual_callout`
- `timeline`
- `comparison`
- `data_story`
- `closing`

Rules:

- choose the simplest layout that supports the content
- do not use decorative layout labels that provide no rendering value
- keep layout names stable across plan and render stages
