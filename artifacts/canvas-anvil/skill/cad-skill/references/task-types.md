# CAD Task Types

## `plan`

Use for:

- requirement analysis
- design strategy
- room program
- spatial assumptions
- zoning logic

Standard artifact:

- `cad_plan`

## `2d_svg`

Use for:

- new floorplan drafting
- redraw requests
- localized edits to the 2D floorplan

Standard artifacts:

- `cad_patch`
- full SVG

## `bom`

Use for:

- material schedules
- fixture lists
- procurement-ready BOM outputs

Standard artifact:

- `cad_bom`

## `images`

Use for:

- analysis boards
- renovation drawing boards
- render image sets

Standard artifacts:

- dedicated analysis-board prompts
- master renovation scheme JSON
- seven dedicated sheet prompts
- local image files
